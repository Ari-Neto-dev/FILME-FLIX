# FILME-FLIX

Aplicação web para busca e visualização de filmes utilizando consumo de API externa.

O projeto permite visualizar lista de filmes, detalhes, avaliações e informações adicionais de cada título.

## Tecnologias utilizadas

- JavaScript
- HTML5
- CSS3
- API REST

## Funcionalidades

- Listagem de filmes
- Busca de filmes por título
- Visualização de detalhes do filme
- Interface simples e responsiva

## Objetivo do projeto

Projeto desenvolvido para prática de integração com APIs e construção de interfaces web focadas em experiência do usuário.
